# Config
Windows Configuration
=====================

This is a very old software, maybe I started the development long before Windows XP or even Windows 2000,
I forgot. It must be before 1998.

I used this software to change some settings in the registry where there was no GUI in the control panel.
Newer versions of windows do not work with this settings, therefore I highly recommend not to save any settings.

The last Windows version, I changed the logon background with this software was Windows 7.
